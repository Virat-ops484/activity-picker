# مُنتقي الأنشطة

[View full project on microbit\.org](https://microbit.org/ar/projects/make-it-code-it/activity-picker)

To code this project yourself, right click on the background of the workspace and choose ‘delete all blocks’, then follow the coding video below. You can find the `forever` and `on start` blocks in the `Basic` section.

### ما هذا؟

هل تجد صعوبة في إتخاذ قرار أو الموافقة على ما ينبغي عليك عمله؟ دع برنامج micro:bit يختار لك!

### ماهي طريقة عمله ؟

* الضغط على الزر A يجعل micro:bit يختار **رقمًا عشوائيًا** بين 1 و 6.
* يقوم بتخزين الرقم في **متغير**  يسمى random_number.
* يختبر البرنامج الرقم العشوائي باستخدام **الخيارات**. تبعًا للعدد المختار، يتم عرض أنشطة مختلفة.
* لا يختبر على وجه التحديد إذا كان الرقم العشوائي 6 - لماذا هذا؟
