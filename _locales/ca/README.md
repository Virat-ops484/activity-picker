# Selector d'activitat

[View full project on microbit\.org](https://microbit.org/ca/projects/make-it-code-it/activity-picker)

To code this project yourself, right click on the background of the workspace and choose ‘delete all blocks’, then follow the coding video below. You can find the `forever` and `on start` blocks in the `Basic` section.

### En què consisteix?

Trobes difícil decidir o acordar el que fer? Deixa que aquest programa de micro:bit esculli per tu!

### Com funciona

* En prémer el botó A, la teva micro:bit tria un **nombre aleatori** entre 1 i 6.
* Emmagatzema el nombre en una **variable** anomenada "random_number" (nombre_atzar).
* El programa prova el nombre aleatori mitjançant la **"selecció"**. Segons el nombre, es mostren diferents activitats.
* Observa que no es prova si el nombre aleatori és 6, per què deu ser això?
